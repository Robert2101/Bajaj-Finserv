# Bajaj Finserv Health - SRM Full Stack Challenge

## Live Links
Frontend: https://bajaj-finserv-zeta-navy.vercel.app 
Backend API: https://bajaj-finserv-7npw.onrender.com

## Tech Stack
- Node.js
- Express.js
- React (Vite)
- Render
- Vercel

## API Endpoint

POST /bfhl

Request:
{
  "data": ["A->B", "A->C", "B->D"]
}

## Features
- Input validation
- Duplicate edge detection
- Tree hierarchy generation
- Cycle detection
- Depth calculation
- Summary statistics

## Run Locally

### Backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
